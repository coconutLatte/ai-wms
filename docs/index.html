📦 What It Does

📊 Admin Dashboard

React + Ant Design web UI for warehouse managers — inventory visibility, order management, real-time dashboards.

📱 PDA Operations

React mobile web app for warehouse operators — barcode scanning, receiving, picking, putaway, cycle counting.

⬇️ Hardware Integration

Downward adapters for WCS (conveyors, sorters) and RCS (AGV/AMR robots) — task dispatch, status feedback.

⬆️ Business Integration

Upward adapters for MES (production orders) and ERP (purchase/sales orders) — order sync, inventory reconciliation.

🧬 Self-Evolution Engine

📖 Roadmap→ 🎯 Pick Task→ 🤖 Claude Code→ ✅ Tests→ 📝 Commit→ 🚀 Push

Triggered every hour by cron. Lean roadmap: max 10 pending tasks. Auto-discovers new work when running low.

🚀 Quick Start

git clone git@github.com:coconutLatte/ai-wms.git
cd ai-wms
make setup          # Install deps, start PostgreSQL + Redis
make build          # Build all services
make test           # Run tests
make run-admin      # Admin API → http://localhost:8080
make run-pda        # PDA API  → http://localhost:8081

📂 Project Structure

Go monolith — DDD layered (domain → service → repository → api) with PostgreSQL and Redis.

frontend/
